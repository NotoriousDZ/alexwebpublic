Value Proposition

ALX serves as a community based currency to facilitate the transcations of in game skins.

  • Fast transaction times.
  • 100% transparent ( all transaction are visible on bscscan.com)
  • Fully Automated and Trustless Smart Contract.
  • Low transaction fees. ($0 fee for buying, 0.1% for selling.)
  • Open sourced API for onboarding game developers.
  • Pegged exchange rate to BNB tokens.
